# 🌸 Sakura Strike — Anime Monster Siege

A single-file, keyboard-only anime third-person shooter. Defend the shrine, fight yokai, crack cases, and survive the horde. No install, no build step — just open `index.html` in a browser.

## Play

Open `index.html` in Chrome, Safari, or any modern browser. That's it. Progress saves automatically to your browser.

## Controls

| Key | Action |
| --- | --- |
| WASD | Move |
| ◀ ▶ | Turn · ▲ ▼ tilt/aim |
| J / F | Fire (hold for auto) |
| E (hold) | Scope |
| V | Toggle first/third person |
| K / L | Punch / Kick (combos) |
| Q / I / C | Abilities · ultimate |
| Space | Dodge roll |
| R | Reload (active-reload window) |
| 1–9, 0 | Hotbar (10 slots: weapons & items) |
| B | Shop · T inventory · O settings · X open chest |
| Esc | Pause |

## Features

- **7 modes** — World campaign, Battleground, Boss Rush, Street Scavenge, Today's Special (rotates daily), Aim Training, plus difficulty (Easy/Medium/Hard)
- **4 battlegrounds** — Sakura Shrine, Neon City, Oni Volcano, Mega City
- **8 weapon classes** across 5 rarities, with special abilities
- **Cases** — weapon, treasure, and ability cases bought with in-game crystals
- **Skins** with passive powers, perks, gun designs, and a 10-slot hotbar
- **AI squad lobbies**, smart enemies with their own abilities, and random wave events

Built with [Three.js](https://threejs.org/). One HTML file, ~130 KB.
